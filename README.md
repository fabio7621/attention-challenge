# 🎯 注意力挑戰計時器

一個幫助你訓練專注力的網頁應用程式，支援挑戰模式和輕鬆訓練模式。

## ✨ 功能特色

### 🏆 挑戰模式

- **嚴格計時**：一旦按下暫停，本次計時結束並記錄成績
- **最高紀錄追蹤**：自動記錄你的最長專注時間
- **歷史記錄**：保存最近 10 筆專注記錄
- **視覺化進度**：每 30 秒點亮一格，超過 5 分鐘格子變綠色
- **數據持久化**：使用 Cookie 儲存，關閉瀏覽器也不會丟失

### 🌿 輕鬆訓練模式

- **彈性暫停**：可以隨時暫停和繼續，不會結束計時
- **純練習用**：不計入紀錄和最高分
- **低壓力**：適合剛開始練習專注力的使用者

## 📖 使用方法

### 開始使用

1. **開啟網頁**：將 HTML 檔案用瀏覽器打開
2. **選擇模式**：點擊上方的「挑戰模式」或「輕鬆訓練」標籤
3. **閱讀提醒**：注意左側紅色提醒框「任何使自己注意力飄散掉都要重新計算」

### 挑戰模式使用

```
1. 點擊「開始專注 ✨」按鈕
   ↓
2. 開始計時，按鈕變成「暫停 ⏸」
   ↓
3. 保持專注，看著進度條逐漸點亮
   ↓
4. 注意力飄散時，點擊「暫停 ⏸」
   ↓
5. 本次記錄自動儲存到歷史列表
```

**重要規則**：

- ⚠️ 一旦暫停就無法繼續，會直接結束本次計時
- 🏆 如果打破最高紀錄，會自動更新
- 📜 歷史記錄最多保留 10 筆

### 輕鬆訓練模式使用

```
1. 點擊「開始練習 🌿」按鈕
   ↓
2. 開始計時，按鈕變成「暫停 ⏸」
   ↓
3. 需要休息時點擊「暫停 ⏸」
   ↓
4. 準備好了點擊「繼續 ▶️」
   ↓
5. 可以重複暫停和繼續
```

**特點**：

- ✅ 可以無限次暫停/繼續
- 🚫 不會儲存記錄
- 🎯 適合探索自己的專注極限

## 🎨 視覺化說明

### 進度條顏色

| 狀態                | 顏色     | 說明           |
| ------------------- | -------- | -------------- |
| 未開始              | 灰色     | 尚未點亮的格子 |
| 專注中 (0-5 分鐘)   | 紫色漸層 | 正常專注狀態   |
| 專注中 (5 分鐘以上) | 綠色漸層 | 達成里程碑！   |

### 計時顯示

- **大數字**：當前計時（分:秒）
- **金色區塊**：最長專注時間（時:分:秒）
- **歷史記錄**：顯示日期時間和持續時長

## 💾 資料儲存

應用程式使用兩個 Cookie 來儲存資料：

- `focusHistory`：歷史記錄列表（最多 10 筆）
- `focusBest`：最高紀錄（單一數值）

**注意**：

- 資料儲存在瀏覽器本地
- 清除瀏覽器 Cookie 會導致資料遺失
- 輕鬆訓練模式的資料不會被儲存

## 🎯 進階技巧

### 如何提升專注時間

1. **從短時間開始**：先設定 5 分鐘目標
2. **消除干擾源**：關閉手機通知、社交媒體
3. **使用輕鬆模式練習**：先熟悉自己的專注節奏
4. **逐步增加難度**：適應後再使用挑戰模式

### 什麼情況算「注意力飄散」？

- 📱 查看手機
- 💬 回覆訊息
- 🌐 瀏覽無關網頁
- 💭 長時間發呆或走神
- ☕ 離開座位休息

### 合理使用建議

- ✅ 每次專注後適度休息
- ✅ 記錄自己的專注時段規律
- ✅ 設定實際可達成的目標
- ❌ 不要過度追求數字而忽略品質
- ❌ 感到疲累時不要勉強

## 📝 常見問題

**Q: 歷史記錄可以保存多久？**  
A: 只要不清除瀏覽器 Cookie，可以永久保存（最多 10 筆）。

**Q: 可以匯出資料嗎？**  
A: 目前版本暫不支援，資料僅保存在瀏覽器本地。

**Q: 為什麼切換模式後計時會重置？**  
A: 這是為了避免兩個模式的資料混淆，確保記錄的準確性。

**Q: 輕鬆模式的資料會保存嗎？**  
A: 不會，輕鬆模式純粹用於練習，不會儲存任何記錄。

**Q: 可以刪除單筆歷史記錄嗎？**  
A: 目前只能一次清除全部歷史記錄，最高紀錄不會被清除。
